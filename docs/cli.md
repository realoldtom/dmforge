## `dmforge deck build`
```shell
Usage: dmforge deck build [OPTIONS]                                                                                                                                                                                                  
                                                                                                                                                                                                                                      
 Generate a spell deck from the SRD with optional filters.                                                                                                                                                                            
                                                                                                                                                                                                                                      
╭─ Options ──────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────╮
│ --output                                TEXT     Output filename for the deck. [default: deck.json]                                                                                                                                │
│ --limit           -n                    INTEGER  Limit the number of spells. [default: None]                                                                                                                                       │
│ --summarize           --no-summarize             Summarize spell descriptions. [default: no-summarize]                                                                                                                             │
│ --summary-length                        INTEGER  Max length of summary in characters. [default: 250]                                                                                                                               │
│ --class                                 TEXT     Comma-separated class filter (e.g. wizard,cleric). [default: None]                                                                                                                │
│ --level                                 TEXT     Comma-separated spell level(s) (e.g. 1,2,3). [default: None]                                                                                                                      │
│ --school                                TEXT     Comma-separated spell school(s). [default: None]                                                                                                                                  │
│ --interactive                                    Select specific spells interactively after filtering.                                                                                                                             │
│ --help                                           Show this message and exit.                                                                                                                                                       │
╰────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────╯
```

 Generate DALL·E art for each spell card in a deck.                                                                                                                                                                                   
                                                                                                                                                                                                                                      
╭─ Arguments ────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────╮

## `dmforge deck render`
```shell
Usage: dmforge deck render [OPTIONS] DECK_FILE                                                                                                                                                                                       
                                                                                                                                                                                                                                      
 Render a deck to PDF or HTML.                                                                                                                                                                                                        
                                                                                                                                                                                                                                      
╭─ Arguments ────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────╮
│ *    deck_file      PATH  Deck JSON file to render. [default: None] [required]                                                                                                                                                     │
╰────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────╯
╭─ Options ──────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────╮
│ --format                              TEXT     Output format (pdf or html). [default: pdf]                                                                                                                                         │
│ --layout                              TEXT     Layout: 'sheet' (6/page) or 'cards' (1/page). [default: sheet]                                                                                                                      │
│ --output                              TEXT     Output file path. [default: None]                                                                                                                                                   │
│ --theme                               TEXT     CSS theme to apply. [default: default]                                                                                                                                              │
│ --debug                                        Also save raw HTML output.                                                                                                                                                          │
│ --summarize         --no-summarize             Summarize before rendering. [default: no-summarize]                                                                                                                                 │
│ --summary-length                      INTEGER  Max summary length. [default: 250]                                                                                                                                                  │
│ --help                                         Show this message and exit.                                                                                                                                                         │
╰────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────────╯
```
